# 桌面应用开发指南

## 可行的技术方案（macOS）

### 方案1: PyQt6 / PySide6 ⭐⭐⭐⭐⭐

**推荐指数：** ⭐⭐⭐⭐⭐（最推荐）

**技术栈：**
- Qt框架 + Python绑定
- 跨平台GUI框架

**优点：**
- ✅ 原生macOS风格
- ✅ 功能非常强大
- ✅ 界面美观、可定制性强
- ✅ 支持拖拽、多线程、进度条等
- ✅ 有丰富的组件库
- ✅ 可以打包成 `.app` 文件

**缺点：**
- ⚠️ 学习曲线稍陡（需要1-2周）
- ⚠️ 需要额外安装依赖
- ⚠️ 打包后体积较大（~50MB）

**复杂程度：** 中等（3/5）

**开发时间：** 2-3天（基础界面）

**依赖安装：**
```bash
pip install PyQt6
pip install pyinstaller  # 用于打包
```

---

### 方案2: Gradio ⭐⭐⭐⭐

**推荐指数：** ⭐⭐⭐⭐（最简单）

**技术栈：**
- Python + Web界面
- 自动生成UI

**优点：**
- ✅ 极简开发（几行代码）
- ✅ 界面美观现代
- ✅ 自动生成UI组件
- ✅ 支持实时预览
- ✅ 非常适合AI/工具类应用

**缺点：**
- ⚠️ 需要浏览器访问
- ⚠️ 不是传统桌面应用
- ⚠️ 无法打包成独立 `.app`

**复杂程度：** 非常简单（1/5）

**开发时间：** 1小时

**依赖安装：**
```bash
pip install gradio
```

## 打包成 .app 文件

### PyQt6 → .app

使用 PyInstaller：

```bash
pip install pyinstaller

pyinstaller --windowed \
  --name "HTML2Markdown" \
  --icon icon.icns \
  --add-data "requirements.txt:." \
  gui_app.py
```

输出：`dist/HTML2Markdown.app`

### Gradio/Streamlit

无法直接打包成传统 `.app`，但可以：

1. **使用 py2app**
```bash
pip install py2app
# 创建setup.py后
python setup.py py2app
```

2. **创建启动脚本**
```bash
#!/bin/bash
cd "$(dirname "$0")"
python3 -m streamlit run app.py
```

保存为 `.command` 文件，双击运行
