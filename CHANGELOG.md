  - 自动处理嵌套目录的路径关系

### 改进
- 📝 更新 README 文档，添加output目录使用说明

## v1.1.0 - 2025-11-13

### 新增功能
- ✨ 支持自动识别和提取文章中的图片和视频资源
- ✨ 支持下载图片和视频到本地（使用 `--download` 或 `-d` 参数）
- ✨ 自动创建资源文件夹（`文章标题_files/`）
- ✨ 显示资源检测和下载进度

### 问题修复
- 🐛 修复包含空格的文件路径导致图片无法显示的问题
  - 使用 Markdown 标准的 `<>` 语法包裹包含空格的路径
  - 示例：`![](<文件夹 包含空格/image.jpg>)`

### 改进
- 📝 更新 README 文档，添加详细的使用说明
- 📝 添加常见问题解答（FAQ）
